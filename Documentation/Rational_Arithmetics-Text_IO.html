Rational_Arithmetics.Text_IO

This package provides text input and output for rational numbers.

Specification

with Ada.Text_IO;
use  Ada.Text_IO;

package Rational_Arithmetics.Text_IO is

  procedure Get (Item: out Rational);
  procedure Put (Item: in  Rational);

  procedure Get (File: in File_Type; Item: out Rational);
  procedure Put (File: in File_Type; Item: in  Rational);

end Rational_Arithmetics.Text_IO;

Put and Get work similarly to the conventional operations in Text_IO for numbers.

If the denominator is 1, put writes the number in the format "sn", else writes it in the format "sn/m" where there is no white space around '/', and the sign 's' is blank for non-negative numbers, n is nonnegative and m positive.

Get skips any white space, then reads the numerator. If no '/' follows, the denominator is 1 and reading stops. Else the operator is skipped and the denominator is read; the result is reduced. Data_Error is raised if either the numerator or the denominator is illegal.

Examples:

The rational number 6/3 will be output as " 2", the number -3/6 will be output as "-1/2".

Reading 1E3/25 will result in 40/1.

Reading 4*3 will stop after the 4, resulting in 4/1; the rest will stay in the input stream.
