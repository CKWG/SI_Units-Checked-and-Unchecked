SI.Vector_Space.Transformation

This package defines matrices for special rotations.

Specification

generic

package Generic_SI.Generic_Vector_Space.Generic_Transformation is

  function Rotation (Axis: Vector; By: Angle) return Matrix;

  function Rotation (Yaw, Pitch, Roll: Angle := 0.0 * "rad") return Matrix;

end Generic_SI.Generic_Vector_Space.Generic_Transformation;

A rotation of a vector V via an orthogonal right-handed matrix M (det M = 1) is achieved by transforming a vector V with the following equation:

  V' := M * V

Note that the vector is rotated, not the coordinate system.

1. Rotation (right-handed) around an axis given by a vector.

  function Rotation (Axis: Vector; By: Angle) return Matrix;

Axis must not be a null vector and Angle must be in radians.

2. Successive rotation (right-handed) around the three coordinate axes.

  function Rotation (Yaw, Pitch, Roll: Item := 0.0 * Radian) return Matrix;

The rotations are performed right to left so that this transformation M fulfills the equation

  M (Yaw, Pitch, Roll) = M (Yaw) * M (Pitch) * M (Roll)

where Roll is the first rotation around the x-axis, Pitch the second around the y-axis (rotation) and Yaw the third around the z-axis.
