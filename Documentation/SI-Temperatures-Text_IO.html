SI.Temperatures.Text_IO

This package enables text input and output for Celsius degrees.

Get and Put read and write the numeric value like standard Text_IO.

The syntax of an external Celsius item is closely based on the one of SI items:

   external_item ::= value unit_factor
   unit_factor   ::= ° identifier
   identifier    ::= letter {letter}

Get: Reading procedes like reading for SI items according to the syntax above. The only allowed identifier is the single character C. Any others will raise Illegal_Unit (there is no discrimination between Unit_Error and Illegal_Unit). (Instead, we could just check whether "°C" follows. In order to be compatible with Get for Item, this is not done.)

Put: If no exception is raised when writing the value, writes the unit °C with no leading space.

Specification

with Ada.Text_IO;
use  Ada.Text_IO;

generic
package Generic_SI.Generic_Temperatures.Generic_Text_IO is

  pragma Elaborate_Body;

  Default_Fore: Field := 2;
  Default_Aft : Field := Real'Digits - 1;
  Default_Exp : Field := 3;

  procedure Get (X    : out Celsius;
                 Width: in  Field := 0);
  procedure Get (File : in  File_Type;
                 X    : out Celsius;
                 Width: in  Field := 0);

  procedure Put (X   : in Celsius;
                 Fore: in Field := Default_Fore;
                 Aft : in Field := Default_Aft;
                 Exp : in Field := Default_Exp);
  procedure Put (File: in File_Type;
                 X   : in Celsius;
                 Fore: in Field := Default_Fore;
                 Aft : in Field := Default_Aft;
                 Exp : in Field := Default_Exp);

  procedure Get (From: in  String;
                 X   : out Celsius;
                 Last: out Positive);
  procedure Put (To  : out String;
                 X   : in  Celsius;
                 Aft : in  Field  := Default_Aft;
                 Exp : in  Field  := Default_Exp);

end Generic_SI.Generic_Temperatures.Generic_Text_IO;

Examples:

4.2*"°C" is output as 4.2°C and can be read as such; reading 4.2*°C raises Illegal_Unit, as does 4.2*K.
