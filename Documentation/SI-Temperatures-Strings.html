SI.Temperatures.Strings

This package provides a sustitute for the homonymous attributes.

For the numeric part, Image and Value behave like the corresponding attributes and may raise the appropriate exceptions.

If no exception is raised, then:

Image appends the unit "°C".

Value checks if the numeric part is followed by the correct unit, else Unit_Error is raised. Trailing blanks are ignored.

Specification

generic

package Generic_SI.Generic_Temperatures.Generic_Strings is

  pragma Elaborate_Body;

  function Image (X: Celsius) return String;
  function Value (X: String ) return Celsius;

end Generic_SI.Generic_Temperatures.Generic_Strings;

Examples

Image (0.1*"°C") = " 1.00000E-01°C"

Value (" 1.00E-01°C   ") = 0.1*"°C"  -- trailing blanks are ignored
Value (" 1.00E-01°C x ")             -- raise Unit_Error
