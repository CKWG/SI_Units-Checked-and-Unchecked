SI.Natural_Constants

This package is where you can add any natural constants you need in your project. Most of the constants change their values with increasing measurement accuracy (except for the very basic ones). Just take the latest values.

SI Redefinition was fixed on November 16, 2018, in Versailles; however, the new definition became effective only on May 20, 2019: World Metrology Day. As of this date, six SI base units are now defined via fixed natural constants (only the Candela has not been redefined).

More information on the definition of the seven base units is to be found here:

  • Meter
  • Kilogram
  • Second
  • Ampere
  • Kelvin
  • Mole
  • Candela

The numeric values of the newly fixed natural constants are from the CODATA 2017 special adjustment. They were calculated from data available before the 1st of July 2017. See ArXiv, Table 1.

Specification

generic

package Generic_SI.Generic_Natural_Constants is

  -- Speed of Light in Vacuum

  Speed_of_Light: constant Speed := 2.997_924_58E+8*"m/s";  -- exact (by definition)
  c: Speed renames Speed_of_Light;

  -- Particles

  Elementary_Charge: constant Charge := 1.602_176_634E-19*"C";  -- exact (by definition)
  e_e: Charge renames Elementary_Charge;

  Electron_Mass: constant Mass := 9.109_382_6E-31 *"kg";
  Proton_Mass  : constant Mass := 1.672_621_71E-27*"kg";

  m_e: Mass renames Electron_Mass;
  m_p: Mass renames Proton_Mass;

  -- Permittivity and Permeability of Vacuum

  Mu_0 : constant Item := 4.0E-7*Pi*"kg*m/C**2";  -- exact; c = 1/Sqrt(Eps_0*Mu_0)
  Eps_0: constant Item := 1.0/(Mu_0*c**2);        -- exact; 8.854_187_817E-12 F/m

  -- Quantum Mechanics

  Planck: constant Item := 6.626_070_15E-34*"J*s";  -- exact (by definition)
  h     :          Item renames Planck;
  h_bar : constant Item := h / (2.0*Pi);

  Alpha: constant Item := e_e**2/(2.0*Eps_0*h*c);  -- exact; 1/137.035_999_11

  -- Thermodynamics

  Boltzmann: constant Item := 1.380_649E-23*"J/K";     -- exact (by definition)
  Avogadro : constant Item := 6.022_140_76E+23/"mol";  -- exact (by definition)

  k_B: Item renames Boltzmann;
  N_A: Item renames Avogadro;

  Gas_Constant: constant Item := N_A * k_B;  -- exact; 8.314 J/(mol*K)

  -- Black body radiation

  Stefan_Boltzmann: constant Item := (2.0*Pi**5/15.0)*(k_B/h)**3*(k_B/c**2);  -- careful with underflow
  Sigma           :          Item renames Stefan_Boltzmann;  -- exact; 5.670E-8*W/(m**2*K**4)

  -- Gravity

  Gravity: constant Item := 6.674_2E-11*"m**3/(kg*s**2)";

end Generic_SI.Generic_Natural_Constants;

I didn't find a better abbreviation for the elementary charge than e_e, normally it's named just e, but this name conflicts with the Euler number e. e_e stands for eelementary or eElectron.
