Rational_Arithmetics.Strings

This package provides replacements for the 'Image and 'Value attributes.

Specification

package Rational_Arithmetics.Strings is

  pragma Elaborate_Body;

  function Image (X: Rational) return String;
  function Value (X: String  ) return Rational;

end Rational_Arithmetics.Strings;

Since the attributes 'Image and 'Value cannot be defined, corresponding functions are defined. They work similarly to the predefined attributes for the constituent parts.

  function Image (X: Rational) return String;

Image returns the argument as a string. If the denominator is 1, the string has the format "sn", else it has the format "sn/m" where there is no white space around '/', and the sign 's' is blank for non-negative numbers, n is nonnegative and m positive.

  function Value (X: String) return Rational;

Value turns the argument into a (reduced) rational number ignoring any leading or trailing blanks. It raises Constraint_Error if the string does not have the aforementioned format.
